# BatchTransfer UI

> [!WARNING]
> **This repository is deprecated and no longer maintained.**
>
> It has been superseded by [**1001-digital/batch-transfer**](https://github.com/1001-digital/batch-transfer) — an overhauled rewrite with a new UI and an expanded feature set: move many **ERC-721**, **ERC-1155**, or **ERC-20** tokens in a single transaction, to one wallet or to many at once.
>
> 👉 **Head to [github.com/1001-digital/batch-transfer](https://github.com/1001-digital/batch-transfer) for the current version.** This repo is kept around for historical reference only.
